Steps to Write First Java Application

Writing your first Java program requires installing the JDK, selecting an editor, writing the code, saving, compiling with javac, and running with java.

1

Install JDK

Download JDK from Oracle. Set PATH env variable to JDK/bin folder.

2

Select Editor

Use Notepad, Notepad++, EditPlus for learning. Use Eclipse/NetBeans in real-time.

3

Write Program

Write Java code using predefined API (Application Programming Interface).

4

Save File

Save as ClassName.java. If class is public, filename must match class name.

5

Compile

Run javac Test.java → generates Test.class (bytecode).

6

Run

Run java Test (no .class). JVM loads and executes the bytecode.

File Naming Rules

  • If file contains a public class/interface/enum, the filename must match that public element's name.
  • Only one public class is allowed per .java file.
  • If no public element, any filename works — but using the main() class name is recommended.
public class A { } // File must be named A.java class FirstApp { public static void main(String[] args) { System.out.println("First Java Application"); } } // Two public classes = Compilation Error public class A { } public class B { } // ERROR

Switching JDK Versions with Batch Files

If multiple JDK versions are installed, the first one in PATH is used. Create .bat files to switch quickly:

:: java8.bat set path=C:\Java\jdk1.8.0\bin; D:\> java8.bat -- Java 8 now active -- D:\> java9.bat -- Java 9 now active --
In real-time projects always use an IDE (Eclipse, IntelliJ IDEA, NetBeans) instead of plain text editors.
